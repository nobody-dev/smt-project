# SMT JavaScript project

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
gulp build
```

### After just start index.html in 'build' folder
```
build/index.html
```

### Technology stack:
```
- PUG (ex Jade), modules
- SCSS
- Adaptive
- Javascript
- JQuery
- Self-made gulp
```
